# Node.js and React.js Boilerplate

이 보일러플레이트 코드는 register, login, logout, auth 기능을 포함하고 있습니다.

# 🧑‍💻 Get started

```
npm install

cd client
npm install

//at root direcotry
npm run dev
```

### MongoDB 사용법

1. dev.js file을 config 폴더 안에 생성
2. mongoDB 정보를 dev.js file안에 삽입

```
module.exports = {
    mongoURI: 'your_MongoDB_URI'
}
```
